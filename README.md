# Beginner's Toolkit for Go (Golang) - Moringa GenAI Capstone

A simple CLI tool that fetches random programming jokes using Go's standard library.

## Quick Start
1. Install Go from https://go.dev/dl/.  
2. Clone this repo.  
3. Open in VS Code → Terminal → `cd code` → `go run main.go`.  

Full guide and docs: See `docs/toolkit.md`.
